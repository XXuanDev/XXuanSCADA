# XXuanSCADA

简介
-------------

XXuanSCADA 是一个针对设备运行过程中，实现设备数据的高效收集、实时数据的监控、设备控制、设备运行参数设置的SCADA系统。 

### 技术栈
开发语言：C#
运行环境：.NET Framework 4.8
PLC：西门子 200 SMART
协议：S7
